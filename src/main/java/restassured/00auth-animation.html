Proving Identity

🔑
Credentials (user/pass, token)
🏛️
Auth Server validates
🎫
JWT / Session token issued
✅
Identity confirmed — 200 OK
⬡ Token Issued — Identity Confirmed
✗ 401 Unauthorized — Invalid Credentials
401 — Not authenticated 200 — Authenticated
Authorization — 403 Forbidden

📋

🛡️

🔐

🔓

✗ 403 Forbidden — Insufficient Role

REST Assured + TestNG Code Java 


  


  
  

    

      
Authentication

      
Happens first, before any access

      
Answers: Who are you?

      
Failure → 401 Unauthorized

      
Uses: username/password, API key, JWT, OAuth tokens

      
REST Assured: .auth().basic(), .auth().oauth2()

    

    

      
Authorization

      
Happens after authentication

      
Answers: What can you do?

      
Failure → 403 Forbidden

      
Uses: roles, scopes, permissions, ACLs, policies

      
REST Assured: assert statusCode(403) for wrong roles
