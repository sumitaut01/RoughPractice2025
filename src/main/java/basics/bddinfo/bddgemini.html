Feature

The high-level description of the software feature being tested.

Given

Describes the initial context or pre-condition (the state of the world).

When

Describes the key action or event performed by the user.

Then

Describes the expected outcome or post-condition (assertions).

Scenario Outline

Runs the same scenario multiple times with different data using an Examples table.

Background

Steps that run before every scenario in a single feature file.

Practical Assertions in "Then" Steps

JUnit 5 Example

@Then("the welcome message should be {string}")
public void verifyWelcome(String expectedMsg) {
    String actual = dashboard.getWelcomeText();
    // JUnit 5: (expected, actual, message)
    assertEquals(expectedMsg, actual, "Welcome text mismatch!");
    assertTrue(dashboard.isProfileIconVisible());
}

TestNG Example

@Then("the user should see {int} products")
public void verifyProductCount(int expectedCount) {
    int actual = productPage.getCount();
    // TestNG: (actual, expected, message)
    Assert.assertEquals(actual, expectedCount, "Product count wrong!");
}

Assertions Comparison

JUnit 5 parameters: (Expected, Actual, "Message"). TestNG parameters: (Actual, Expected, "Message").

JUnit 5 (org.junit.jupiter.api.Assertions)

// Basic Equality
assertEquals(expected, actual, "Error message");

// Boolean
assertTrue(condition, "Must be true");
assertFalse(condition, "Must be false");

// Null checks
assertNull(obj);
assertNotNull(obj);

// Collections
assertArrayEquals(expectedArr, actualArr);

// Exceptions
assertThrows(NullPointerException.class, () -> {
    obj.doSomething();
});

// Soft Assertions (Grouped)
assertAll("Group Name",
    () -> assertEquals(1, 1),
    () -> assertTrue(true)
);

TestNG (org.testng.Assert)

// Basic Equality (Actual first!)
Assert.assertEquals(actual, expected, "Error message");

// Boolean
Assert.assertTrue(condition, "Must be true");
Assert.assertFalse(condition, "Must be false");

// Null checks
Assert.assertNull(obj);
Assert.assertNotNull(obj);

// Collections
Assert.assertEquals(actualList, expectedList);

// Soft Assertions
SoftAssert softAssert = new SoftAssert();
softAssert.assertTrue(condition);
softAssert.assertAll(); // Critical call!
