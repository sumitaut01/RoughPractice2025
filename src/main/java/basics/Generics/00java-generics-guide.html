What Are Java Generics?
Write type-safe, reusable code — catch errors at compile time, not runtime.
Core Idea: Generics let you write a class, interface, or method once and use it with any data type — while the compiler enforces type safety for you.

The Problem Without Generics

List list = new ArrayList();
list.add("hello");
list.add(123);             // No error at compile time!

String s = (String) list.get(1);  // ❌ ClassCastException at RUNTIME!

The Fix With Generics

List<String> list = new ArrayList<>();
list.add("hello");
list.add(123);  // ❌ Compile-time error — caught immediately!

Key Benefits

🛡️
Type Safety
Errors caught at compile time, not runtime surprises.
♻️
Reusability
Write once, use with any type — no duplication.
✂️
No Casting
Compiler knows the type — manual casts eliminated.
📖
Readable Code
Intent is clear — List<User> tells the story.

Generic Class
T is a placeholder — plug in the real type at usage time.
// T is a "Type Parameter" — placeholder for a real type
public class Box<T> {
    private T value;

    public void set(T value)  { this.value = value; }
    public  T   get()         { return value; }
}

// Usage — same class, different types
Box<String>  stringBox = new Box<>();
Box<Integer> intBox    = new Box<>();

stringBox.set("Sumit");
intBox.set(42);

System.out.println(stringBox.get()); // Sumit
System.out.println(intBox.get());    // 42

Multiple Type Parameters

public class Pair<K, V> {
    private final K key;
    private final V value;

    public Pair(K key, V value) { this.key = key; this.value = value; }
    public K getKey()   { return key;   }
    public V getValue() { return value; }
}

// Usage
Pair<String, Integer> entry = new Pair<>("powerRank", 5);
QA Use-case: Use generic classes for ApiResponseWrapper<T>, TestContext<T>, and result holders across your framework.
Generic Method
Declare <T> before the return type — makes a method generic independent of the class.
public class Utils {

    // <T> before return type = this method is generic
    public static <T> void printArray(T[] array) {
        for (T item : array) System.out.print(item + " ");
        System.out.println();
    }

    // Generic method with return type
    public static <T> List<T> asList(T... items) {
        return new ArrayList<>(Arrays.asList(items));
    }
}

// Compiler infers T from arguments — no explicit type needed
String[]  names = {"Sumit", "Ravi", "Priya"};
Integer[] nums  = {10, 20, 30};
Utils.printArray(names); // Sumit Ravi Priya
Utils.printArray(nums);  // 10 20 30

Generic Deserializer Method (Framework Pattern)

// Deserialize any JSON into any type — one method for all endpoints
public static <T> T fromJson(String json, Class<T> clazz) throws Exception {
    return new ObjectMapper().readValue(json, clazz);
}

// Usage — T is inferred from the target variable
User        user = fromJson(responseBody, User.class);
PricingRule rule = fromJson(responseBody, PricingRule.class);
Key Rule: The <T> must appear before the return type. The compiler infers T from the argument — no explicit type needed at the call site.
Bounds & Wildcards
Control which types are allowed — essential for flexible but safe generic code.

Upper Bounded — extends

// T must be Number or a subclass (Integer, Double, Float...)
public static <T extends Number> double sum(List<T> list) {
    double total = 0;
    for (T item : list) total += item.doubleValue();
    return total;
}
sum(List.of(1, 2, 3));          // 6.0  ✅
sum(List.of(1.5, 2.5));       // 4.0  ✅

Wildcards

// ? extends T — read only (producer)
public void readNumbers(List<? extends Number> list) {
    for (Number n : list) System.out.println(n);
}

// ? super T — write only (consumer)
public void addNumbers(List<? super Integer> list) {
    list.add(10); list.add(20);
}

// ? — unbounded (any type, no read/write guarantee)
public void printList(List<?> list) {
    for (Object obj : list) System.out.println(obj);
}
PECS = Producer Extends, Consumer Super

Use ? extends T when you only READ. Use ? super T when you only WRITE. Use ? when you do neither.
WildcardCan Read?Can Write?Use When
? extends T✅ Yes (as T)❌ NoReading / iterating a list
? super T⚠️ As Object✅ YesAdding items to a list
?⚠️ As Object❌ NoPrinting, size check
Generic API Response Wrapper
Parse any API response body into any POJO — one class serves all endpoints.
public class ApiResponseWrapper<T> {

    private int    statusCode;
    private T      body;
    private String errorMessage;

    // Static factory — T inferred from the Class argument
    public static <T> ApiResponseWrapper<T> from(
            Response response, Class<T> clazz) {

        ApiResponseWrapper<T> w = new ApiResponseWrapper<>();
        w.statusCode = response.getStatusCode();
        try {
            w.body = new ObjectMapper().readValue(response.asString(), clazz);
        } catch (Exception e) {
            w.errorMessage = e.getMessage();
        }
        return w;
    }

    public int    getStatusCode()   { return statusCode;   }
    public T      getBody()         { return body;         }
    public String getErrorMessage() { return errorMessage; }
}

// ── Usage ────────────────────────────────────────────────
@Test
public void testGetUser() {
    Response response = given().get("/users/1");
    ApiResponseWrapper<User> result = ApiResponseWrapper.from(response, User.class);

    Assert.assertEquals(result.getStatusCode(), 200);
    Assert.assertEquals(result.getBody().getName(), "Sumit");
}
Benefit: One wrapper class serves User, PricingRule, Product, Order — no duplication across API test classes.
Generic Schema Validator
Validate JSON schema AND deserialize into a typed POJO — fluent and chainable.
public class SchemaValidator<T> {

    private final Class<T>    type;
    private final ObjectMapper mapper = new ObjectMapper();

    public SchemaValidator(Class<T> type) { this.type = type; }

    // Validate against a schema file in classpath/schemas/
    public SchemaValidator<T> validateSchema(Response response, String schemaFile) {
        response.then().body(JsonSchemaValidator
            .matchesJsonSchemaInClasspath("schemas/" + schemaFile));
        return this;
    }

    public T deserialize(Response response) throws Exception {
        return mapper.readValue(response.asString(), type);
    }

    // One-shot: validate schema + return typed object
    public T validateAndDeserialize(Response response, String schemaFile)
            throws Exception {
        validateSchema(response, schemaFile);
        return deserialize(response);
    }
}

// ── Usage ────────────────────────────────────────────────
@Test
public void testPricingRule() throws Exception {
    Response response = given().spec(requestSpec).get("/pricing/rules/1");

    PricingRule rule = new SchemaValidator<>(PricingRule.class)
        .validateAndDeserialize(response, "pricing-rule-schema.json");

    Assert.assertEquals(rule.getPowerRank(), 5);
}
Schema files live in src/test/resources/schemas/. The validator auto-prefixes the path — just pass the filename.
Generic Validation Chain
Fluent builder — chain HTTP + body + custom lambda validations on ANY response type.
public class ResponseValidator<T> {

    private final Response   response;
    private final T          body;
    private final SoftAssert softAssert = new SoftAssert();

    private ResponseValidator(Response r, T body) { this.response = r; this.body = body; }

    public static <T> ResponseValidator<T> of(Response r, Class<T> clazz) throws Exception {
        T body = new ObjectMapper().readValue(r.asString(), clazz);
        return new ResponseValidator<>(r, body);
    }

    // HTTP-level checks
    public ResponseValidator<T> statusIs(int expected) {
        softAssert.assertEquals(response.getStatusCode(), expected, "Status mismatch");
        return this;
    }

    public ResponseValidator<T> responseTimeLessThan(long ms) {
        softAssert.assertTrue(response.getTime() < ms, "Slow response: " + response.getTime() + "ms");
        return this;
    }

    // Body field checks via method references
    public <F> ResponseValidator<T> fieldEquals(String name, Function<T,F> fn, F expected) {
        softAssert.assertEquals(fn.apply(body), expected, "Field [" + name + "] mismatch");
        return this;
    }

    public <F extends Comparable<F>> ResponseValidator<T> fieldGreaterThan(
            String name, Function<T,F> fn, F threshold) {
        F actual = fn.apply(body);
        softAssert.assertTrue(actual.compareTo(threshold) > 0,
            "[" + name + "] = " + actual + " not > " + threshold);
        return this;
    }

    // Custom lambda validation
    public ResponseValidator<T> validate(String desc, Predicate<T> condition) {
        softAssert.assertTrue(condition.test(body), "Failed: " + desc);
        return this;
    }

    // Terminal — runs all soft asserts, returns typed body
    public T assertAllAndGet() { softAssert.assertAll(); return body; }
}

// ── Usage ────────────────────────────────────────────────
PricingRule rule = ResponseValidator.of(response, PricingRule.class)
    .statusIs(200)
    .responseTimeLessThan(2000)
    .fieldEquals("ruleName",  PricingRule::getRuleName,  "TIER_DISCOUNT")
    .fieldEquals("powerRank", PricingRule::getPowerRank, 5)
    .fieldGreaterThan("price", PricingRule::getPrice, 0.0)
    .validate("discount 0–100", r -> r.getDiscount() >= 0 && r.getDiscount() <= 100)
    .assertAllAndGet();
Generic Field Extractor
Pull typed fields from any POJO using method references — no casting, no magic strings.
public class FieldExtractor<T> {

    private final T body;
    public FieldExtractor(T body) { this.body = body; }

    // Extract via Function — compile-time safe
    public <F> F get(Function<T, F> extractor) {
        return extractor.apply(body);
    }

    // Extract list of sub-fields
    public <F> List<F> getList(Function<T, List<F>> extractor) {
        return extractor.apply(body);
    }

    // Extract + assert in one shot
    public <F> F getAndAssert(Function<T, F> extractor, F expected, String field) {
        F actual = extractor.apply(body);
        Assert.assertEquals(actual, expected, "Mismatch on [" + field + "]");
        return actual;
    }
}

// ── Usage ────────────────────────────────────────────────
FieldExtractor<PricingRule> fx = new FieldExtractor<>(rule);

String       name = fx.get(PricingRule::getRuleName);
int          rank = fx.getAndAssert(PricingRule::getPowerRank, 5, "powerRank");
List<String> tags = fx.getList(PricingRule::getTags);
Method References: PricingRule::getRuleName IS a Function<PricingRule, String> — compiler infers both type parameters automatically.
Generic Response Comparator
Compare two API responses field by field — regression and diff testing against saved baselines.
public class ResponseComparator<T> {

    private final T          baseline;
    private final T          actual;
    private final SoftAssert softAssert = new SoftAssert();

    public ResponseComparator(T baseline, T actual) {
        this.baseline = baseline; this.actual = actual;
    }

    public <F> ResponseComparator<T> compareField(String name, Function<T,F> fn) {
        softAssert.assertEquals(fn.apply(actual), fn.apply(baseline),
            "Regression: [" + name + "] changed");
        return this;
    }

    public ResponseComparator<T> ignore(String reason) {
        System.out.println("Skipping: " + reason); return this;
    }

    public void assertNoDifferences() { softAssert.assertAll(); }
}

// ── Regression Test ──────────────────────────────────────
@Test
public void regressionTestPricingRule() throws Exception {
    PricingRule baseline = loadFromSnapshot("rule_1_baseline.json", PricingRule.class);
    PricingRule actual   = new ObjectMapper().readValue(
                               given().spec(requestSpec).get("/pricing/rules/1").asString(),
                               PricingRule.class);

    new ResponseComparator<>(baseline, actual)
        .compareField("ruleName",  PricingRule::getRuleName)
        .compareField("powerRank", PricingRule::getPowerRank)
        .compareField("discount",  PricingRule::getDiscount)
        .ignore("createdAt — timestamp changes every run")
        .assertNoDifferences();
}
Baseline snapshots live in src/test/resources/testdata/snapshots/. Save a known-good response once — compare automatically on every run.
Generic Data Provider
Load typed test data from JSON files — one loader for any POJO, wired to @DataProvider.
public class GenericDataProvider {

    // Generic loader — reads JSON array into List<T>
    public static <T> List<T> loadTestData(String filePath, Class<T> clazz)
            throws Exception {
        ObjectMapper mapper = new ObjectMapper();
        return mapper.readValue(
            new File(filePath),
            mapper.getTypeFactory().constructCollectionType(List.class, clazz)
        );
    }

    // Convert List<T> to Object[][] for TestNG
    public static <T> Object[][] toDataProvider(List<T> list) {
        return list.stream()
                   .map(item -> new Object[]{item})
                   .toArray(Object[][]::new);
    }

    @DataProvider(name = "pricingRules")
    public static Object[][] pricingRules() throws Exception {
        return toDataProvider(loadTestData(
            "src/test/resources/testdata/pricing-rules.json", PricingRule.class));
    }
}

// ── Usage ────────────────────────────────────────────────
@Test(dataProvider = "pricingRules", dataProviderClass = GenericDataProvider.class)
public void testPricingRule(PricingRule rule) {
    // rule is strongly typed — no casting needed!
    Assert.assertTrue(rule.getPowerRank() > 0);
}
Generic ThreadLocal Context
Thread-safe storage for parallel test execution — one class powers both WebDriver and RestAssured spec management.
// Generic ThreadLocal holder — works for ANY type T
public class TestContext<T> {
    private final ThreadLocal<T> context = new ThreadLocal<>();

    public void set(T value) { context.set(value);    }
    public T    get()        { return context.get(); }
    public void remove()    { context.remove();      }
}

// ── WebDriver Manager ────────────────────────────────────
public class DriverManager {
    private static final TestContext<WebDriver> ctx = new TestContext<>();

    public static void      setDriver(WebDriver d) { ctx.set(d);        }
    public static WebDriver getDriver()            { return ctx.get(); }
    public static void      removeDriver()         { ctx.remove();     }
}

// ── API Client Manager ───────────────────────────────────
public class ApiClientManager {
    private static final TestContext<RequestSpecification> ctx = new TestContext<>();

    public static void                 setSpec(RequestSpecification s) { ctx.set(s);    }
    public static RequestSpecification getSpec()                       { return ctx.get(); }
}

// ── BaseTest ─────────────────────────────────────────────
@BeforeMethod
public void setUp() {
    DriverManager.setDriver(new ChromeDriver());  // stored per-thread
}

@AfterMethod
public void tearDown() {
    DriverManager.getDriver().quit();
    DriverManager.removeDriver();  // ⚠️ CRITICAL — prevent memory leak!
}
Always call remove() in @AfterMethod! Skipping this causes memory leaks in parallel TestNG thread pools — each thread keeps a reference to the driver forever.
Generic DB Query Helper
Works for PostgreSQL and Oracle — one helper, any return type via functional RowMapper<T>.
// Functional interface — maps one ResultSet row to type T
@FunctionalInterface
public interface RowMapper<T> {
    T map(ResultSet rs) throws Exception;
}

public class DbHelper {

    public static <T> List<T> queryForList(
            Connection conn, String sql, RowMapper<T> mapper) throws Exception {

        List<T> results = new ArrayList<>();
        try (PreparedStatement stmt = conn.prepareStatement(sql);
             ResultSet rs           = stmt.executeQuery()) {
            while (rs.next()) results.add(mapper.map(rs));
        }
        return results;
    }

    public static <T> Optional<T> queryForOne(
            Connection conn, String sql, RowMapper<T> mapper) throws Exception {
        List<T> r = queryForList(conn, sql, mapper);
        return r.isEmpty() ? Optional.empty() : Optional.of(r.get(0));
    }
}

// ── Usage — PostgreSQL ───────────────────────────────────
List<PricingRule> rules = DbHelper.queryForList(
    pgConn,
    "SELECT id, rule_name, power_rank FROM pricing_rules WHERE active = true",
    rs -> new PricingRule(rs.getInt("id"), rs.getString("rule_name"), rs.getInt("power_rank"))
);

// ── Usage — Oracle (identical API!) ──────────────────────
List<Product> products = DbHelper.queryForList(
    oracleConn,
    "SELECT code, price FROM products",
    rs -> new Product(rs.getString("code"), rs.getDouble("price"))
);
Lambda as RowMapper: The lambda IS the RowMapper<T> implementation. Works identically for PostgreSQL and Oracle — just swap the Connection.

@Test
public void fullPricingRuleValidation() throws Exception {

    Response response = given()
                           .spec(ApiClientManager.getSpec())
                           .get("/pricing/rules/1");

    // ① Schema validation + typed deserialization
    PricingRule rule = new SchemaValidator<>(PricingRule.class)
                           .validateAndDeserialize(response, "pricing-rule.json");

    // ② Full validation chain — HTTP + body + custom logic
    PricingRule validated = ResponseValidator.of(response, PricingRule.class)
        .statusIs(200)
        .responseTimeLessThan(2000)
        .fieldEquals("ruleName",  PricingRule::getRuleName,  "TIER_DISCOUNT")
        .fieldEquals("powerRank", PricingRule::getPowerRank, 5)
        .fieldGreaterThan("price", PricingRule::getPrice, 0.0)
        .validate("discount must be 0–100",
                  r -> r.getDiscount() >= 0 && r.getDiscount() <= 100)
        .assertAllAndGet();

    // ③ Field extraction for downstream assertions
    FieldExtractor<PricingRule> fx = new FieldExtractor<>(validated);
    int rank = fx.getAndAssert(PricingRule::getPowerRank, 5, "powerRank");

    // ④ Regression compare against saved baseline
    PricingRule baseline = loadFromSnapshot("rule_1_baseline.json", PricingRule.class);
    new ResponseComparator<>(baseline, validated)
        .compareField("ruleName",  PricingRule::getRuleName)
        .compareField("powerRank", PricingRule::getPowerRank)
        .ignore("createdAt — timestamp")
        .assertNoDifferences();
}
